# The Namada SDK

The Namada software development kit (SDK) can be found in the `namada` repo under the path [`namada/shared`](https://github.com/anoma/namada/tree/main/shared). The SDK is written in Rust and can be used to interact with the Namada blockchain, by constructing transactions, signing them, and submitting them to the network.

## Installation

The Namada SDK can be installed by creating a new Rust project and adding the following to the `Cargo.toml` file:
